# Physical AI & Humanoid Robotics Textbook

This is the online textbook for the "Physical AI & Humanoid Robotics" course, built with [Docusaurus](https://docusaurus.io/).

## Installation

```bash
cd book
npm install
```

## Local Development

```bash
npm start
```

This command starts a local development server and open up a browser window. Most changes are reflected live without having to restart the server.

## Build

```bash
npm run build
```

This command generates static content into the `build` directory and can be served using any static content hosting service.

## Deployment

The website is deployed to GitHub Pages. Refer to `.github/workflows/deploy.yml` for CI/CD configuration.
